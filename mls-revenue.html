Case Study: MLS Revenue & Superstar Economic Impact

Executive Summary & Deliverables

This econometric study evaluates the financial impact of superstar acquisitions in Major League Soccer, specifically focusing on Inter Miami CF. Utilizing a panel dataset from 2018 to 2024, the analysis employs fixed-effects regression models to isolate demand shocks, accounting for market size, team performance, and Covid-19 disruptions.

Panel Data Structure

The analysis relies on a multi-year panel dataset tracking franchise performance, attendance, and market conditions.

Feature Name Data Type Description
Revenue_real Float Inflation-adjusted franchise revenue (Dependent Variable)
miami2023 / miami2024 Binary Treatment variables isolating the superstar arrival
PctCapacity Float Stadium capacity utilization constraint
MarketSize Float Regional market size index
Points / Playoff Int/Binary Controls for team athletic performance

Econometric Modeling (Stata)

Core estimation code from mls_revenue_regression.do — fixed-effects regression with standard errors clustered at the team level. 

// Generate Logarithmic Revenue
gen ln_Revenue_real = ln(Revenue_real)

// Main Fixed-Effects Model with Clustered Standard Errors
reg ln_Revenue_real ///
    miami2023 miami2024 ///                     // Superstar treatment effects
    i.team_id i.Year ///                        // Team and Year Fixed Effects
    Points Playoff MarketSize PctCapacity, ///  // Athletic & Market Controls
    vce(cluster team_id)                        // Clustered Standard Errors

// Robustness Check: Excluding Covid-19 Anomaly Year (2020)
reg Revenue_real ///
    miami2023 miami2024 ///
    i.team_id i.Year ///
    Points Playoff MarketSize PctCapacity ///
    if Year != 2020, ///
    vce(cluster team_id)
