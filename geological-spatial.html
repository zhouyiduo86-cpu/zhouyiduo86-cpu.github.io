Case Study: Geological Spatial Database & Exploration Mapping

Executive Summary

This Geographic Information Systems (GIS) project transformed historical, unstructured survey records into a highly accurate spatial database. The resulting cartographic deliverables isolate key structural features and highlight localized mineral occurrences (Zinc, Lead, and Barium) across the Northwest Territories RED claims area.

Download Exploration Map

Spatial Data Architecture

The spatial database was constructed in ArcGIS Pro utilizing strict georeferencing protocols. The mapped layers include:

Layer Data Type Description
Mineral Occurrences Point Data Symbolized locations of Zn, Pb, and Ba showings.
Structural Features Polyline Digitized fault lines and bedding orientations.
Lithological Units Polygon Geologic boundaries including dolomite, limestone, and argillaceous formations.
Coordinate System Spatial Reference Applied precise georeferencing grid for exploration accuracy.
